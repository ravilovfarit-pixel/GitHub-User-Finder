# GitHub-User-Finder
GUI-приложение «GitHub User Finder» с использованием GitHub API, сохранением избранных пользователей и Git.
Автор Хайрутдинов Эмир
Описание Приложение на Python для поиска профилей Github через официальный API
как использовать API  Используется эндпоинт https://github.com{query}
Ключ API в базовой версии не требуется (лимит  10 запросов в минуту для неавторизованных)
