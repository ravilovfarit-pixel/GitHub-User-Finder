 Weather Diary (Дневник погоды)

Автор: Хайрутдинов Эмир Эдуардович

 Описание программы
Приложение для ведения личного дневника погоды. Позволяет фиксировать дату, температуру, описание погодных условий и наличие осадков. Данные сохраняются в формате JSON, что позволяет просматривать историю записей при повторном запуске.

 Функциональные возможности
- Добавление новых записей с валидацией данных.
- Просмотр всех записей в удобной таблице.
- Фильтрация записей по конкретной дате.
- Фильтрация записей по температуре (выше указанного значения).
- Автоматическое сохранение и загрузка данных из файла `weather_data.json`.
